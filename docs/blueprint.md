# **App Name**: TaskZen

## Core Features:

- Task List View: Add, view, and manage tasks in a list format. Each task includes name, due date, urgency, required hours, and tags. Users should be able to assign a color for each tag.
- Task Calendar View: Display tasks in a calendar format. Allows creation of tasks directly from the calendar. Updates bi-directionally with list view.
- Task Filtering and Sorting: Filter tasks by tags or sort by due date, urgency, or required hours.
- Chatbot Task Parser: A chatbot feature that parses a text list of tasks and automatically generates task list objects.
- Chatbot Task Advice: A chatbot feature that gives advice on how to organize and plan for tasks.
- Panic Mode: A 'Panic Mode' feature that analyzes the user's schedule, including tasks, due dates, and urgency levels, and provides a tool for completing as many tasks as possible on time.

## Style Guidelines:

- Primary color: Dark grey (#333333) to maintain the black theme.
- Background color: Black (#000000) to provide a minimalistic black theme.
- Accent color: Electric blue (#7DF9FF) for highlights and interactive elements.
- Body and headline font: 'Inter' for a modern, neutral feel.
- Use minimalistic, line-based icons.
- Clean and spacious layout with a focus on readability.
- Subtle transitions and animations to provide feedback and enhance user experience.
